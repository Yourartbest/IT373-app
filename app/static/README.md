# app/static

Place app-specific static assets here (served as-is):

- images/: product photos, icons, logos, screenshots
- css/: optional page-specific stylesheets
- js/: optional scripts

Tip: keep filenames lowercase, hyphenated, and web-friendly (no spaces).
